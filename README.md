# TicTacToe (Morpion) — Projet Python

Jeu de **morpion (TicTacToe)** développé en **Python** dans le cadre d’un projet personnel.  
Ce projet a pour objectif de mettre en pratique les bases de la programmation : logique, conditions, boucles et fonctions.

Dépôt GitHub :  
https://github.com/GA2207/TicTacToe

---

## Fonctionnalités

- Jeu de morpion sur une grille **3×3**
- **Deux joueurs** (tour par tour)
- Détection automatique :
  - des victoires (lignes, colonnes, diagonales)
  - des matchs nuls
- Affichage de l’état de la grille à chaque tour

---

## Prérequis

- **Python 3.8 ou plus**

Vérifier la version installée :
```bash
python --version


## Installation

git clone https://github.com/GA2207/TicTacToe.git
cd TicTacToe


## Lancer le jeu

python Projets-tic-tac-toe/main.py



## Principe du Jeu

- Le jeu affiche une grille de morpion
- Les joueurs jouent à tour de rôle
- Chaque joueur entre une position valide
- La partie se termine lorsqu’un joueur gagne ou qu’il n’y a plus de coups possibles
